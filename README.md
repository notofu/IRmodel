# Requirement

* numpy 1.17.5
* japanize-matplotlib 1.1.3
* matplotlib 3.3.4
* glob
* cloudpickle 1.6.0
* music21 6.7.1

# Usage

実行方法
```bash
cd IRmodel-main
python3 main.py
```

パラメータ
``` Python: main.py
maximum_symbol_num: 各音符に付与される最大シンボル数(1 ~ 3)
num_of_symbols: シンボルの種類(8または16)
```
